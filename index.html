Overview

AIGCleaner is an AI-powered text processing platform designed for content creators, academic researchers, and enterprises. It provides APIs for AIGC detection, intelligent text rewriting, and style enhancement. Visit the official platform at aigcleaner.com to explore the full suite of tools.

Core Features

AIGC Detection

Analyze text to identify AI-generated content patterns with high accuracy. Returns a confidence score and detailed segment-level breakdown.

Text Rewriting

Restructure AI-generated text to achieve a natural, human-like writing style while preserving the original meaning and context.

Academic Enhancement

Improve academic papers with proper citation formatting, terminology consistency, and discipline-specific tone adjustments.

Batch Processing

Process large volumes of text efficiently through the batch API. Supports async operations with webhook callbacks.

Quick Start

Get started with AIGCleaner in minutes. First, create an account at aigcleaner.com and obtain your API key from the dashboard.

Installation

pip install aigcleaner-sdk

Basic Usage

from aigcleaner import AIGClient

client = AIGClient(api_key="your-api-key")

# Detect AI-generated content
result = client.detect(text="Your text content here...")
print(f"AI probability: {result.score:.2%}")

# Rewrite text for human-like quality
rewritten = client.rewrite(
    text="Your AI-generated text...",
    mode="natural",   # Options: natural, academic, creative
    strength=0.7
)
print(rewritten.text)

API Endpoint Reference

Method Endpoint Description
POST /api/v1/detect Detect AIGC content
POST /api/v1/rewrite Rewrite text
POST /api/v1/enhance Academic enhancement
GET /api/v1/usage Check API usage stats
